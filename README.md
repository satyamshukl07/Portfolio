# My Portfolio Website

A clean and professional personal portfolio website built with pure HTML, CSS, and JavaScript.

## Live Demo
🔗 [View Portfolio](https://yourusername.github.io/my-portfolio)

## Sections
- **Hero** — Introduction and call-to-action
- **About** — Bio, photo, and stats
- **Skills** — Technical skills with tags
- **Projects** — Featured project cards with links
- **Resume** — Download button and highlights
- **Contact** — Contact links and message form

## How to Use

1. Clone or download this repo
2. Open `index.html` in any browser
3. Replace placeholder content with your real info:
   - Your name, bio, and stats in `index.html`
   - Add `assets/images/profile.jpg` (your photo)
   - Add `assets/resume/your-resume.pdf`
   - Update project screenshots in `assets/images/projects/`
   - Update links (GitHub, LinkedIn, email) in `index.html`

---

## Deploy on GitHub Pages (Free Hosting)

1. Push this project to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/root`
4. Your site will be live at `https://yourusername.github.io/repo-name`
